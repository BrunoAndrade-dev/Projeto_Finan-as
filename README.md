# Sistema para análise de dados sobre educação financeira

### Sobre o projeto

[cite_start]Este repositorio contem o backend e frontend do sistema desenvolvido para o plano de trabalho de iniciacao cientifica focado na avaliacao de ferramentas de analise de dados para educacao financeira no estado do Maranhao[cite: 10, 11]. [cite_start]O projeto esta vinculado a Universidade Estadual do Maranhao (UEMA) e ao Departamento de Engenharia de Computacao[cite: 1, 3, 5].

### Objetivos principais

[cite_start]O trabalho busca avaliar tecnologias de analise de dados para subsidiar o desenvolvimento de um sistema que utilize aprendizado de maquina na previsao da economia estadual[cite: 47]. As metas especificas incluem:

* [cite_start]Pesquisar ferramentas voltadas para educacao financeira[cite: 50].
* [cite_start]Avaliar as melhores solucoes para a implementacao do sistema[cite: 51].
* [cite_start]Construir e testar um prototipo funcional em tempo real[cite: 52].
* [cite_start]Gerar contribuicoes academicas como artigos e patentes[cite: 54].

### Metodologia e tecnologias

[cite_start]A fundamentacao tecnica do backend utiliza estatistica multivariada e machine learning para transformar dados brutos em conhecimento acionavel[cite: 63]. [cite_start]O ecossistema integra fontes de dados publicas e institucionais como Serasa Experian, Portal da Transparencia e IBGE[cite: 61].

Os eixos fundamentais de processamento sao:

* [cite_start]Analise descritiva e exploratoria para diagnostico do letramento financeiro[cite: 64].
* [cite_start]Analise de agrupamento para identificacao de perfis de endividamento por regiao ou idade[cite: 64, 65].
* [cite_start]Desenvolvimento de indicadores para monitoramento da saude financeira da populacao[cite: 65].

### Estrutura tecnologica

Conforme definido para esta etapa da plataforma:

* ETL: Python
* Backend: Java com Spring Boot
* Banco de dados: PostgreSQL

### Contexto de pesquisa

[cite_start]O projeto justifica-se pela necessidade de combater o superendividamento e promover o bem-estar economico[cite: 33]. [cite_start]Em 2026, o cenario de inadimplencia no Brasil afeta mais de 80 milhoes de pessoas, gerando impactos diretos na saude mental e na produtividade laboral[cite: 27, 28]. [cite_start]Este sistema atua como um vetor estrategico para a resiliencia socioeconomica local[cite: 33, 43].
